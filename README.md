## Executar

Para executar o programa utilizas os seguintes comandos:
```bash
make jogo
```

```bash
./jogo
```

## Testes

O projeto utiliza a biblioteca CUnit para realizar os testes às funções.

```bash
make testar
```

```bash
./testar
```
